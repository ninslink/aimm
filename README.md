# AIMM website (GitHub Pages)

This is a static website hosted on GitHub Pages.

## Publisher view counts (weekly)
The Publications page displays publisher-side view counts for Frontiers articles.
A GitHub Action runs weekly (Mondays 06:00 UTC) and updates:

- `assets/frontiers_metrics.json`

If you fork/clone this repo, ensure GitHub Actions are enabled and that the workflow has permission to write to the repository (`Settings → Actions → General → Workflow permissions → Read and write`).

# AIMM Website

This repository hosts the website for the **Artificial Intelligence for Monitoring Malnutrition (AIMM)** project.

## Pages
- Home
- Project
- Team
- Updates
- Publications
- Contact

## Publications
- Publication metadata is imported from Crossref based on the DOIs listed in `assets/pubs.json`.
- Frontiers view counts are stored in `assets/frontiers_metrics.json` and updated weekly via GitHub Actions.

Enable workflow write permissions:
Settings → Actions → General → Workflow permissions → Read and write.
